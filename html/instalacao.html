Programação básica com Python Instalação

Este projeto pretende colocar aqui uma introdução básica à Linguagem de Programação Python

Instalação do ambiente de desenvolvimento Python

Como citado antes, Python está disponível para várias Sistemas Operacionais, dentre eles Linux, Unix, Mac OS X e Windows.

Há muitos meios de instalação, para cada Sistemas Operacional, mas abordaremos a instalação clássica utilizado o instalador baixado diretamento do site do projeto Python, versões para Windows e para Mac OS X; No caso do Linux mostraremos como instalar nas distribuições Debian, CentOS e Arch Linux. Cada uma delas possui pacotes pré-compilados do ambiente Python, bem como sistema de instalação própria.
